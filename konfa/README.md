
# Учебный конфигурационный язык -> TOML

## Описание
Инструмент командной строки для трансляции учебного конфигурационного языка в TOML.
Используется синтаксический анализатор **Lark**.

## Возможности
- Числа
- Массивы
- Константы
- Постфиксные выражения (+, -, *, /, max)

## Установка
```
pip install lark
```

## Использование
```
python tool/main.py --input examples/finance.conf
```

## Тесты
```
pytest
```

## Примеры
- Финансовая конфигурация
- Игровая конфигурация
